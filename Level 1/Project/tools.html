MY CODING TOOLS

These are some fun resources that help kids like me to learn coding. They make learning a lot more easy and exciting!

Visual Studio Code

This is a super cool program I use to write real code. It looks like a notebook but for typing code!

I used it to make a website with HTML and CSS (yes, this website). It helps me fix mistakes easily too, especially since this is all new to me.

Why I like it: It feels like I’m a real coder and it has colorful words!

CSS Diner

CSS Diner is a fun game that teaches us how to pick the right parts of a webpage to style using CSS. We play by choosing the right “plates” on the table!

Why kids love it: It feels like a yummy food game that helps you learn how to make websites look nice.

Scratch

Scratch is a colorful coding tool where we can drag and snap blocks to make our own games and stories. We don’t need to actually type code, it is just about putting pieces together to create your own story. It's like playing with building blocks!

Why it’s fun: We can create personal animations and games, and share them with friends. It feels like magic when the characters start moving!
